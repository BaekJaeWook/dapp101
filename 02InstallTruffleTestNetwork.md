# Truffle 설치와 Test network 설치

Truffle은 이더리움 디앱 프레임워크입니다.
[트러플 홈페이지](https://truffleframework.com)

# Truffle 설치
프레임워크인 트러플을 설치합니다.
트러플을 사용하기 위해서는 nodejs 5.0이상이 설치되어 있어야 합니다.

```
npm install -g truffle
```

정상적으로 설치됐는지 확인하기 위해서 트러플 버전을 출력해봅니다
```
truffle --version
```
